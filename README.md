# \# Loan Approval Prediction System

# 

# A machine learning web application that predicts whether a loan application is likely to be approved based on applicant details.

# 

# \## Features

# 

# \* Data preprocessing and cleaning

# \* Exploratory Data Analysis (EDA)

# \* Logistic Regression model

# \* Random Forest model

# \* Model performance comparison

# \* Loan approval prediction through a Flask web interface

# \* Input validation

# \* Responsive user interface

# 

# \## Dataset

# 

# The project uses a loan approval dataset containing applicant information such as:

# 

# \* Gender

# \* Marital Status

# \* Dependents

# \* Education

# \* Self Employment Status

# \* Applicant Income

# \* Coapplicant Income

# \* Loan Amount

# \* Loan Amount Term

# \* Credit History

# \* Property Area

#
